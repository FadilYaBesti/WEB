Font Style

CSS dapat mengatur beberapa style dari text yang ada pada halaman web yang dibuat. Terdapat beberapa property yang dapat digunakan untuk mengatur style text yaitu sebagai berikut.

  • text align : digunakan untuk mengatur style text yaitu left, center, right.
  • color : digunakan untuk memberikan warna pada text.
  • letter-spacing : mengatur jarak antar kata
  • word-spacing : mengatur jarak antar kata
